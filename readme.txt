== Description ==

Plugin installs affiliate code of Installus.net

== Installation ==

There are three easy steps to get the most value from the Installus links.

= To install the plugin: =
1. Install the Installus links plugin using the WordPress Plugin Installer [here](https://wordpress.org/plugins/installus-links/).
1. Activate the plugin through the “Plugins” menu in your WordPress Dashboard.
1. That’s it! Once activated, your links will automatically be localized for your international audience.

= To connect your installuslink account: =
1. If you don’t have an account already, create a installus.net account [here](https://installus.net/index.php?register=true).
1. After signing in, navigate to the Sources page to create a new Site.
1. Access the Installus links Settings through the “Settings” tab within your WordPress dashboard.
1. Copy the SiteID into your Installus links plugin.
